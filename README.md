1)Cat vs Dog Image Classification using CNN

This project is my implementation of a Convolutional Neural Network (CNN) for classifying images of cats and dogs. The model was developed and trained in Google Colab using TensorFlow and Keras. Through this project, I learned how CNNs extract visual features from images and perform image classification tasks.

2)Project Objective

The goal of this project is to build a deep learning model that can automatically identify whether an input image belongs to a cat or a dog.

3)Tools and Technologies

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* Google Colab

4)Model Architecture

The CNN model consists of multiple convolution and max-pooling layers followed by fully connected dense layers. These layers help the model learn important visual patterns from the training images and make accurate predictions.

5)Project Workflow

1. Load and preprocess image data
2. Build the CNN architecture
3. Train the model on the dataset
4. Evaluate model performance
5. Predict classes for new images

6)Files Included

* `CatVsdog.ipynb` – Complete notebook containing data preprocessing, model training, evaluation, and predictions.
* `README.md` – Project documentation.

7)What I Learned

* Image preprocessing techniques
* CNN architecture design
* Model training and evaluation
* Deep learning workflows using TensorFlow and Keras
* Working with image datasets in Google Colab
